# 🏀 Basketball Scoreboard

A vanilla JavaScript scoreboard to track HOME and GUEST scores in a basketball game.

**Live Demo** → [https://basketballscoreboardjsproject.netlify.app/]

---

## Features

- +1, +2, +3 scoring buttons for each team
- Winner declared at 15 points
- Score card turns green (winner) and red (loser)
- Digital LCD-style font

---

## Built With

- HTML | CSS | JavaScript (no frameworks)

---

## What I Learned

- DOM manipulation
- Event handling with `onclick`
- Managing state with variables
- Dynamic styling with `style.backgroundColor`

---

## Run Locally

Clone the repo and open `index.html` in your browser — that's it.
